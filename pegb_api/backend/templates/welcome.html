This is an example API that will provide responces to the following requests:

GET /users | To retrieve users from the pegb_db database container
GET /posts | To retrieve posts from the pegb_db database container

POST /add | For addition of x and y elements
POST /addmany | For addition of n number of elements (experimental)
POST /subtract | For subtraction of x and y elements
POST /multiply | For multiplication of x and y elements
POST /divide | For division of x and y elements

Example:
$ curl -d '{ "x":5, "y":10 }' -H 'Content-Type: application/json' -X POST http://172.16.0.95:5000/add
{ "Message": 15, "Status Code": 200 }
